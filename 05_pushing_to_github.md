# Pushing to GitHub

Pushing sends your local commits to your GitHub repository.

## Steps:
1. On GitHub, create a new empty repository  
2. Copy the HTTPS URL  
3. In your terminal, connect your local repo to GitHub:

```
git remote add origin https://github.com/yourname/yourrepo.git
```

4. Push your project:
```
git push -u origin main
```

Your code is now online.
