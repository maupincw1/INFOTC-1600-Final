# Making Commits

A commit is a saved snapshot of your project.

## Steps:
1. Save your file changes  
2. Stage the changes:
   ```
   git add .
   ```
3. Commit the changes:
   ```
   git commit -m "Describe what you changed"
   ```

You can repeat these steps each time you update your project.
