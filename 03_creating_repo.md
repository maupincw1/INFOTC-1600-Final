# Creating Your First Repository

A Git repository (repo) is a folder tracked by Git.

## Steps:
1. Create a new folder on your computer  
2. Open Git Bash (Windows) or Terminal (Mac/Linux)  
3. Navigate into that folder  
4. Run:
   ```
   git init
   ```
5. Create or add files to the folder  
6. Check Git’s status:
   ```
   git status
   ```

You now have a Git repository.
